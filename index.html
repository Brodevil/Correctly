üöÄ Installation Steps

Step 1: Download Extension Files

All extension files are in your current Replit project folder. Download or clone the entire project to your local computer.

Extension files: manifest.json, popup.html, popup.js, content.js, background.js, styles.css, generated-icon.png

Step 2: Open Chrome Extensions Page

  1. Open Google Chrome
  2. Navigate to chrome://extensions/
  3. Enable "Developer mode" (toggle in top-right corner)

Step 3: Load the Extension

  1. Click "Load unpacked" button
  2. Select the folder containing all the extension files
  3. The extension should now appear in your extensions list

Step 4: Configure API Keys

When you first install the extension, a setup page will automatically open:

  1. Enter your OpenAI API key (starts with "sk-")
  2. Enter your Gemini API key (starts with "AIza")
  3. Click "Save Keys & Continue"

You can also configure keys later by clicking the extension icon in your toolbar.

üìñ How to Use

Step 1: Open a Google Form

Navigate to any Google Form with multiple choice questions (MCQs).

Step 2: Activate the Extension

  1. Click the extension icon in your toolbar
  2. Click the "Start Auto-Answer" button
  3. The extension will begin processing questions

Step 3: Review Results

Questions will be highlighted:

  • üü¢ Green border - Both AIs agreed, answer selected automatically
  • üü° Yellow border - AIs disagreed, question skipped (review manually)
üí° Pro Tip: Always review skipped questions (yellow highlighted) manually before submitting the form. These are questions where ChatGPT and Gemini gave different answers, indicating potential ambiguity or complexity.

üîß Troubleshooting

Extension Not Working?

  • Make sure you're on a Google Forms page (docs.google.com/forms)
  • Check that both API keys are saved and showing green status
  • Open browser console (F12) to check for any error messages
  • Try reloading the extension from chrome://extensions/

API Errors?

  • Verify your API keys are valid and have available credits
  • Check your OpenAI account billing at platform.openai.com
  • Ensure your Gemini API key is active at Google AI Studio

‚öôÔ∏è Technical Details

AI Models Used:

  • OpenAI: GPT-4o-mini (optimized for speed and cost)
  • Google: Gemini 1.5 Flash (fast, efficient responses)

How It Works:

  1. Extension scans the Google Form for multiple choice questions
  2. Sends each question to both ChatGPT and Gemini simultaneously
  3. Compares the responses from both AI models
  4. If both models give the same answer (A, B, C, or D), it auto-selects that option
  5. If models disagree, the question is skipped for manual review
